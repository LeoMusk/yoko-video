# Timeline and Style Guide

Use these defaults unless the user provides a different platform, brand, or visual direction.

## Timeline Patterns

### 30s Authority Clip

Use for keynote, podcast, interview, livestream, founder talk, public speech.

```text
0-3s    Hook: why this clip matters
3-9s    Reframe the core shift
9-16s   Explain the mechanism
16-23s  Explain what ability becomes valuable
23-30s  Opinion / implication / CTA
```

Preserve original audio. Let the source speaker provide credibility.

### 20s Demo Clip

Use for product demos or screen recordings.

```text
0-2s    Result-first hook
2-7s    Show the artifact/outcome
7-14s   Explain the steps
14-20s  Why it matters / who should care
```

Use fewer cards and more visible source video.

### 45s Explainer Clip

Use when the source contains several distinct claims.

```text
0-3s    Hook
3-12s   Claim 1
12-22s  Claim 2
22-32s  Claim 3
32-42s  Synthesis
42-45s  CTA / branded close
```

## Layout Defaults

### Vertical Stack

Best first-pass layout for most clips:

- Source video: top 45-55% of canvas
- Cards: start around 60-70% vertical position
- Cards may overlap source captions to replace them with higher-value interpretation
- Bottom 12-15% remains safe for platform UI, captions, and descriptions

### Full-Bleed Overlay

Use for emotionally strong footage or dramatic hooks:

- Source video fills canvas
- Add dark vignette
- Place a large hook above platform safe area
- Avoid covering faces

### PiP Explanation

Use when the card is the primary content:

- Card fills most of the canvas
- Source video becomes a small speaker/proof window
- Good for dense concepts, charts, or step-by-step explanation

## Copy Rules

- Use short, concrete hooks.
- Avoid generic AI phrases like "颠覆认知", "不是 X 而是 Y", "未来已来", unless quoting or intentionally parodying.
- Prefer "这对谁意味着什么" over abstract summaries.
- Use one core noun phrase per card: "派 AI", "工作流", "执行层", "数字员工".
- Keep title text large enough for a phone: 72-100px on 1080x1920.
- Keep detail text around 34-46px.

## Visual Style Defaults

Use a high-contrast tech editorial style:

- Background: near black `#05070b` or `#0d1117`
- Text: `#ffffff`, secondary `#d7dde8`
- Accents: rotate per card (`#76ff03`, `#60a5fa`, `#ff2e88`, `#fbbf24`)
- Use no more than one accent color per card
- Avoid excessive gradients or decorative shapes
- Use hard rectangular cards, not soft marketing cards

## Platform Safety

For TikTok, Douyin, Reels, Shorts:

- Avoid key text in the bottom 12-15% of the canvas.
- Keep top labels small; platform profile UI may cover top-left/top-right areas depending on app.
- Center important text between 55-82% of canvas height.
- If the platform will show a title/description below the video, move the main card up and let it cover source subtitles.
